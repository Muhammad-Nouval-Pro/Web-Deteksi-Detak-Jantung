                                Heart Rate Monitoring System
Deskripsi

Heart Rate Monitoring System merupakan sistem monitoring detak jantung berbasis Internet of Things (IoT) yang menggunakan ESP32 dan Pulse Heart Sensor untuk membaca nilai BPM (Beat Per Minute) secara realtime. Data hasil pembacaan sensor dikirim melalui jaringan WiFi ke server dan disimpan ke database PostgreSQL, kemudian ditampilkan pada website monitoring berbasis Next.js.

Sistem ini juga dilengkapi dengan OLED SSD1306 yang berfungsi untuk menampilkan hasil pembacaan BPM secara langsung pada perangkat.

Fitur
Monitoring detak jantung secara realtime
Tampilan BPM pada OLED SSD1306
Pengiriman data menggunakan ESP32 dan WiFi
Penyimpanan data ke PostgreSQL Database
Dashboard monitoring berbasis website
Riwayat pemeriksaan detak jantung
Grafik monitoring BPM
Manajemen sesi pemeriksaan
Teknologi yang Digunakan
Hardware
ESP32 Dev Module
Pulse Heart Sensor
OLED SSD1306
Breadboard
Kabel Jumper
